# Univers App

Une application sociale futuriste avec IA Gemini et Supabase.

## Configuration

### Prérequis
- Node.js 18+
- npm ou yarn

### Installation

```bash
npm install
```

### Variables d'environnement

Créez un fichier `.env.local` avec vos clés:

```env
GEMINI_API_KEY=votre_cle_gemini
```

### Lancer en local

```bash
npm run dev
```

## Technologies

- React 19 + TypeScript
- Vite
- Google Gemini AI
- Supabase
